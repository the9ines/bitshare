üöÄ bitshare App Mockup - Server Setup

Choose one of the following methods to run the bitshare mockup locally:

Method 1: Python 3 (Recommended)

Open Terminal and run:

cd /Users/oberfelder/Desktop/the9ines.com/bitshare/mockup
python3 -m http.server 8000

Then open: http://localhost:8000

Method 2: Node.js

If you have Node.js installed:

cd /Users/oberfelder/Desktop/the9ines.com/bitshare/mockup
npx http-server -p 8000

Then open: http://localhost:8000

Method 3: PHP

If you have PHP installed:

cd /Users/oberfelder/Desktop/the9ines.com/bitshare/mockup
php -S localhost:8000

Then open: http://localhost:8000

Method 4: Custom Python Server

Run the custom server:

cd /Users/oberfelder/Desktop/the9ines.com/bitshare/mockup
python3 simple-server.py

The server will automatically try ports 8000-8009

Method 5: Visual Studio Code

If you have VS Code with Live Server extension:

  1. Open the mockup folder in VS Code
  2. Right-click on index.html
  3. Select "Open with Live Server"

Troubleshooting

Port already in use? Try a different port:

python3 -m http.server 8001
# or
python3 -m http.server 8080

Python not found? Try:

python -m http.server 8000

Permission denied? Make sure you're in the right directory:

ls -la index.html
